Cyber Security ជាអ្វី?៖

Cyber Security គឺជាវិទ្យាសាស្ត្រនិងការអនុវត្តន៍ដែលមានគោលបំណងការពារ ប្រព័ន្ធទិន្នន័យ និងបណ្តាញពីការរួមបញ្ចូល និងការវាយប្រហារដែលមិនមានសិទ្ធិចូលប្រើ។ វាគ្របដណ្តប់ពីការពារជំហាននានាដែលអ្នកប្រើប្រាស់ និងអាជីវកម្មអាចធ្វើដើម្បីរក្សាទុកសុវត្ថិភាពនៃទិន្នន័យឬកម្មវិធីចំពោះការប្រែប្រួលដោយគ្មានការអនុញ្ញាត ឬការវាយប្រហារពីអ្នកចោលឬមនុស្សមិនស្មោះត្រង់។ Cyber Security មានភាពសំខាន់ក្នុងការរក្សាទុកសុវត្ថិភាពឯកសារបុគ្គល និងអាជីវកម្ម, ពីការចោទប្រកាន់ជាច្រើន ដូចជាការលួចទិន្នន័យ ឬការព្យាករណ៍ការវាយប្រហារពីហាក់បីមិនត្រូវបានគ្រោះថ្នាក់។ វាបង្កើតកំណត់ត្រាផ្នែកទាំងអស់ដែលអាចជួយរក្សាឱ្យការប្រើប្រាស់ទិន្នន័យសុវត្ថិភាព និងមិនអនុញ្ញាតឲ្យមានការចូលប្រើប្រាស់ដែលខុសពីសិទ្ធិ, ដូចជាការការពារពីការចូលប្រើប្រាស់ដោយមិនសមត្ថភាព, ការបំផ្លាញព័ត៌មាន, ឬការបំរែបំរួលទិន្នន័យ។ Cyber Security គ្របដណ្តប់ការពារពីវិបត្តិឆ្លងកាត់បណ្តាញ, ការវាយប្រហារផ្សេងៗ, ការចោលការពារជំរុញឲ្យមានការកសាងប្រព័ន្ធផ្តល់សេវាកម្មដែលមានសុវត្ថិភាព និងច្បាស់លាស់ក្នុងការគ្រប់គ្រងទិន្នន័យនិងប្រព័ន្ធបច្ចេកវិទ្យា។
