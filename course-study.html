Front End Web Development

Front-end web development ពាក់ព័ន្ធនឹងការបង្កើតទិដ្ឋភាពដែលមើលឃើញ ដែលមានភាពទាក់ទាញនៃគេហទំព័រដោយប្រើ HTML, CSS និង JavaScript ដើម្បីធានាបាននូវបទពិសោធន៍អ្នកប្រើប្រាស់យ៉ាងរលូន។

Back End Web Development

Back-end web development សំដៅលើ ខាង server-side ឬ Admin website ដែលទទួលខុសត្រូវខាង ផ្ទុកទិន្នន័យ , រៀបចំទិន្នន័យ និង ផ្តល់អ្វីៗទៅខាង client-side ជាដើម។

Full Stack JavaScript

Full stack JavaScript developers គឺមានតម្រូវការខ្ពស់។ ពួកគេដឹងពីរបៀបធ្វើការជាមួយ ផ្នែក Frontend និង Backend និងផ្នែក servers នៃគេហទំព័រ Full Stack តាមរយៈ JavaScript។

Python Development

Python គឺជាភាសាសរសេរកម្មវិធីដែលមានលក្ខណ:បែប high-level, general-purpose។ វាត្រូវបានប្រើសម្រាប់កម្មវិធីបណ្តាញផ្សេងៗគ្នាជាច្រើន Web Applications, Servers, Al, វីដេអូហ្គេម, និង Robot។
